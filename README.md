# John Dorion - Personal Portfolio

My personal portfolio and resume website showcasing my work in cybersecurity, cloud infrastructure, and Linux systems.

**Live site:** https://0x6a64.github.io/resume/

## Tech Stack

- [Astro](https://astro.build)
- [TailwindCSS](https://tailwindcss.com/)
- [DaisyUI](https://daisyui.com/)

Built using the [Astrofy](https://github.com/manuelernestog/astrofy) template.

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```

## Deployment

Automatically deployed to GitHub Pages via GitHub Actions on push to `main`.
